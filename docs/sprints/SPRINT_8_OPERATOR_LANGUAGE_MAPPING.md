# Sprint 8 Operator Language Mapping

| Old wording | New wording |
|---|---|
| Managed Disabled Accounts | Disabled User Accounts |
| No Tracked Jira Site Access | Users Without Jira Access |
| Inactive Without Site Access | Inactive Users (No Access) |
| Bitbucket Only / No Tracked Jira | Bitbucket-only Users |
| Trend lookback | Snapshot history depth |
| Snapshots in Lookback | Available snapshots |
| Rising Risk Sites | Sites with Increasing Risk |
| Warning / Critical Streaks | Sites Repeatedly Flagged |
| Rising Unresolved Sites | Sites with Growing Backlog |
| Trend History | Historical View |
| Atlassian area | Where to look in Atlassian |
| Project sample | Project sample |
| Users source / Managed source | Operator-facing monitoring status |

## Guiding rule
If a non-technical stakeholder cannot understand the phrase in under five seconds, it should be rewritten.
