# Stakeholder Q&A Pack v2

## Is JOM read-only?

Yes. The current release candidate is designed as a read-only operational console.

## Does JOM replace Atlassian administration?

No. JOM highlights issues, routes operators to relevant evidence and supports decision-making. Direct administration remains in Atlassian tools.

## Are reports source-backed?

Yes. The current HTML, JSON and CSV report outputs are generated from existing JOM runtime/static data and validated endpoints.

## Are PDF and PowerPoint exports available?

Not yet. PDF and PowerPoint should remain future work unless specifically implemented later.

## Is this production-ready?

It is a production-candidate baseline suitable for controlled demo and review. Formal deployment, security and resilience hardening remain next steps.
