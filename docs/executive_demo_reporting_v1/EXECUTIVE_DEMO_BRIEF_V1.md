# JOM Executive Demo Brief v1

## Purpose

This brief supports a stakeholder demonstration of the current JOM production-candidate baseline.

## Key Message

JOM is a read-only Atlassian operational intelligence platform. It provides a live Command Centre, Estate Operations Hub, Admin Intelligence Centre and Site Workspace using source-backed runtime/operator data.

## Suggested Demo Flow

1. Open Command Centre and explain estate health, runtime status, alert posture and discovery queue.
2. Open Estate and show monitored/discovered visibility and the Site Detail Panel.
3. Open a Site Workspace from Estate and show single-site investigation.
4. Open Admin and show discovery governance, named access intelligence, runtime/source state and API visibility.
5. Explain compatibility policy: `/api/data`, `/api/source-state` and `/api/site-registry` remain retained while the frontend uses the consolidated workspace model.
