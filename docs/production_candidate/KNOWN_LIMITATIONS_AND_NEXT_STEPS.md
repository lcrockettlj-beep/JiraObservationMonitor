# Known Limitations and Next Steps

- JOM is currently read-only.
- Export controls can remain placeholders unless a dedicated export pack is applied.
- Admin actions are surfaced as governance/intelligence views, not write-back controls.
- Site Workspace uses currently available source-backed datasets.
