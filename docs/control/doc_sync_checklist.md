# Documentation Sync Checklist

After ANY backend change, confirm:

## Runtime
- [ ] /api/data still returns valid payload
- [ ] new fields visible
- [ ] removed fields confirmed intentionally
- [ ] runtime snapshot updated 

## Mechanics manuals
- [ ] Python manual updated if logic changed
- [ ] JSON contract updated if structure changed
- [ ] New modules documented

## Change log
- [ ] change recorded in mechanics_change_log.md

## Optional
- [ ] governance docs updated if scope/value changed
